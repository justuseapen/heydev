HeyDev Widget - Script Tag Test

This page demonstrates the widget auto-initializing from a script tag.

How It Works

The widget reads configuration from the script tag's data attributes:

<script
  src="widget.js"
  data-api-key="your-api-key-here"
  data-endpoint="http://localhost:3000"
></script>

data-api-key (required): Your API key for authentication

data-endpoint (optional): Custom backend URL (defaults to https://api.heydev.io)

Public API

The widget exposes window.HeyDev with these methods:

// Open the feedback panel
window.HeyDev.open();

// Close the feedback panel
window.HeyDev.close();

// Check if panel is open
window.HeyDev.isOpen();

// Get widget version
window.HeyDev.version;

// Destroy widget completely
window.HeyDev.destroy();
